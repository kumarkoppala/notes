:q! -> quit without saving
:wq -> write and quit
:set nu -> number

:/<word-to-search> -> forward search
:?<word-to-search> -> backward search
:<line-number> -> cursor moves to line-number

:%d -> all content delete
:%s/<word-to-search>/<word-to-replace>/g -> entire file all occurences replace
